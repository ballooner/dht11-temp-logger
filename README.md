# How to run
```console
make
sudo ./logger
```

# Required Hardware
* 1 Raspberry Pi 5
* 1 DHT11 Sensor with built-in pull-up resistor

# Hardware Setup
Connect DHT11 + to a power pin
Connect DHT11 OUT to a GPIO pin
Connect DHT11 - to a ground pin

# Software Setup
Download [lgpio](https://abyz.me.uk/lg/download.html) and follow the installation guide
